# Shibboleth APC Paper Draft

Neutral LaTeX workspace for the Shibboleth paper.

## Build

```bash
latexmk -pdf main.tex
```

## Notes

- Uses `article` class on purpose so the content can stabilize before switching to a venue template.
- Section text is seeded from the existing markdown draft and repo specs.
- `refs.bib` is only a starter bibliography.
